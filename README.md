# Trybesmith

Este repositório contém o projeto "_Trybesmith_", requisito em atendimento ao módulo de Back-end - TypeScript, da Trybe. O projeto consiste em desenvolver todas as camadas da aplicação e realizar as operações básicas de um CRUD.

### Habilidades:
 - Declarar variáveis e funções com tipagens Typescript;
 - Construir uma API Node Express utilizando o Typescript.
